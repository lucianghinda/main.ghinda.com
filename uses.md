## Building software

- **[Ruby](https://www.ruby-lang.org)** — my language of choice since 2007. I
  reach for it first, and I write all my scripts in it, including the small LLM
  tools I build.
- **[Ruby on Rails](https://rubyonrails.org)** — what I use to build web apps,
  end to end from the product conversation to the deploy.
- **[Avo](https://avohq.io)** — my main go-to for building admin panels and
  dashboards on top of Rails.
- **[SQLite](https://www.sqlite.org)** — my default database; it takes me
  surprisingly far.
- **[PostgreSQL](https://www.postgresql.org)** — when a project needs more than
  SQLite can give.
- **[Jekyll](https://jekyllrb.com)** — the static-site generator behind this
  blog.

## Editor and terminal

- **[Neovim](https://neovim.io)** — my main editor and IDE; I live in it all
  day. My setup is public:
  [neovim-ruby-rails-config](https://github.com/lucianghinda/neovim-ruby-rails-config).
- **[Ghostty](https://ghostty.org)** — my everyday terminal.
- **[cmux](https://cmux.com)** — a Ghostty-based terminal I switch to when I'm
  running AI coding agents in parallel.

## AI-assisted development

- **[Claude Code](https://claude.com/claude-code)** — my primary agentic coding
  tool.
- **[Codex](https://openai.com/codex)** — OpenAI's coding agent, also in regular
  rotation.
- **[OpenCode](https://opencode.ai)** — the open-source coding agent I reach
  for.
- **[RubyLLM](https://rubyllm.com)** — one API across LLM providers; what I use
  when I build AI into a Rails product.

## Machines and hosting

- **[macOS](https://www.apple.com/macos/)** — my development machine.
- **Linux** — usually [Ubuntu](https://ubuntu.com) or
  [CentOS](https://www.centos.org) — is what I deploy to.
- **[Hetzner](https://www.hetzner.com)** — where my servers live.
- **[Hatchbox](https://hatchbox.io)** — how I manage and deploy most of my Rails
  apps.
- **[Kamal](https://kamal-deploy.org)** — for the deploys I run container-first.

## Writing and audience

- **[Beehiiv](https://www.beehiiv.com)** — the engine behind my newsletters.
- **[Fastmail](https://www.fastmail.com)** — handles all my email.

## Images and video

- **[OBS](https://obsproject.com)** — for recording video.
- **[Xnapper](https://xnapper.com)** — for taking and prettifying screenshots.
- **[Postcraft Studio](https://postcraftstudio.com)** — for generating the
  images I use in articles and share online.

## Running the sites

- **[Tinylytics](https://tinylytics.app)** — lightweight, privacy-friendly
  analytics.
- **[Scribbles](https://scribbles.page)** — powers
  [notes.ghinda.com](https://notes.ghinda.com), my short-notes blog.

## Staying organized

- **[Fizzy](https://www.fizzy.do)** — the kanban tool where I keep my to-do
  list.
